# my_browser

A new Flutter project.

 A Browser Created With Help of WebView And Shared Prefreance.
 <h1>Its Features:-</h1>
 A browser with <br>
     - Shared Preference And Provider<br>
     - In App WebView<br>
     - Bookmark Functionality<br>
     - reload Site<br>
     - Google Search Engine<br>
     - Go back <br>
     - Go Forward<br>
     - Pull to Refresh<br>
     - Add And Remove Bookmark<br>
     - Home Button<br>
